# Pollen Tracker
## Description
A web application that allows users to search for daily city air and weather conditions. 
* Location-based conditions include: 
    * Pollen Levels
    * Air Quality Index
    * Daily Forecast

## Built With
* HTML
* Bulma/CSS
* JavaScript/jQuery 

## APIs Used
* Moment 
* Tomorrow.io
* Air Quality Open Data Platform
* OpenWeather
